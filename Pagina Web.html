¿Que es HTML? ¿Que es CSS?
En resumen, HTML5 sirve como referencia para el software relacionado con el desarrollo de diferentes versiones de páginas web. Define la estructura básica y el código (llamado HTML) que se utiliza para definir el contenido web como texto, imágenes, videos , etc. En resumen CSS es un lenguaje de marcado dedicado a definir, crear y mejorar la representación de documento basados en HTML.

¿Que es un vínculo? ¿Que es una tabla?
Los enlaces o hipervínculos, también llamados hipertextos son los textos o los objetos sobre los que podemos hacer clic para que nos lleven a otra parte del documento, a otra página web en el mismo sitio o a otra página de Internet, entre otras funciones. Las tablas pueden ser consideradas como un grupo de filas donde cada una de ellas contiene un grupo de celdas. Una tabla puede ser insertada en un documento HTML.

Cuarta Semana:

En esta cuarta semana, aprendimos sobre dos tema muy importante, uno de ellos facilita el desarrollo de las páginas web, "los Frameword." Los framework de CSS es una biblioteca de estilos genéricos que puede ser usada para implementar diseños web. Por otro lado vimos las "Librerías", que son un conjunto de códigos de programación.

¿Que son los Frameword? ¿Que son las Librerías?
los Framework sirve para poder escribir código o desarrollar una aplicación de manera más sencilla. Es algo que permite una mejor organización y control de todo el código elaborado, así como una posible reutilización en el futuro. Las Librería es una serie de códigos de programación, un conjunto de código que alguien ha hecho para que tú puedes reutilizar dentro de tus proyectos. El objetivo es sencillo, hacer más fácil y rápido el desarrollo de ciertas funciones dentro de tu app.
